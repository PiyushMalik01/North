# North

**AI-Driven Skill Intelligence & Learning Platform**

Transform learning into skill mastery with AI-driven intelligence and structured progression.

## 🎯 Overview

North is a next-generation learning platform designed specifically for students who want clarity, structure, and real skill development. No more random learning—just guided direction from first semester to first job.

## ✨ Key Features

- **Skill Tree Progression**: Navigate structured learning paths with prerequisite mapping
- **AI-Assisted Learning**: Get personalized roadmaps and contextual guidance
- **Real Assessments**: Prove skills through coding challenges and projects
- **Dynamic Skill Profile**: Build verifiable skill graphs that showcase real capabilities
- **Gamified Engagement**: Track depth, execution, consistency, and collaboration
- **Talent Discovery**: Connect with companies seeking skill-first candidates

## 🚀 Tech Stack

### Frontend
- **Framework**: Next.js 16+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **State Management**: Zustand
- **HTTP Client**: Axios
- **Animations**: Framer Motion
- **Code Editor**: Monaco Editor

### Theme System
- **Dark Mode** (default): Deep navy backgrounds with indigo/purple accents
- **Light Mode**: Clean white backgrounds with optimized contrast
- **Smooth Transitions**: Water-flowing theme switching with ripple effects

- **Product-First**: GitHub × Linear × Notion aesthetic

- **Responsive**: Mobile-first design with tablet/desktop optimization

## 🛠️ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Navigate to frontend
cd frontend

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.
