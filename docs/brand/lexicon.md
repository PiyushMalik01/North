# North — Lexicon ("words from the ice")

The naming system for the platform. One world: **polar, snowy, cold, lit by a north star, guided by Nor.**
Words are short, lowercase, modern — pulled from ice / light / compass / penguin. **No fantasy-gaming words.**

> Convention: zone names are written **lowercase** in UI (`nest`, `trace`, `dive`). Capitalize only at the start of a sentence.

---

## World vocabulary (the constants — appear everywhere)

| word | meaning |
|---|---|
| **north** | the student's goal — their north star ("land a frontend internship") |
| **nor** | the AI guide / companion. always present, never a "zone" |
| **beam** | nor's recommendation — "your beam today →" (the guiding light / next move) |
| **streak** | daily activity streak |
| **freezes** | streak freezes (already perfectly icy) |
| **stats** | the 5 traits — depth, execution, consistency, collaboration, clarity |
| **node** | a single skill node on the path |
| **lume** | the skills you've unlocked (the light you carry) |
| **gleam** | a badge / achievement |
| **breach** | a level-up / breakthrough moment (penguin breaching the surface) |
| **shelf** | saved items / your library |

---

## Zones (the places, mapped to real features)

### the dock — daily core (5)
These are the always-reachable zones in the nav (the "compass dock").

| zone | feature it holds | phase |
|---|---|---|
| **nest** | home / hub — avatar, stats, streak, nor, your beam today | MVP |
| **trace** | skill tree + roadmap — the route north (steps = *nodes*) | MVP |
| **dive** | codespaces IDE — go deep, build & make things | MVP |
| **prove** | assessments — coding / reasoning / rapid-fire (passing marks a node done) | MVP |
| **drift** | the feed — what's moving in your field (items = *signals*) | Phase 2 |

### deeper zones (not daily)

| zone | feature it holds | phase |
|---|---|---|
| **spark** | quests / tasks (team ones = *runs*) | MVP |
| **huddle** | community & chat — the warm gathering place | Phase 2 |
| **flock** | your squads / hubs / crews — your specific people | Phase 3 |
| **flare** | live events — hackathons, contests, college challenges | Phase 2 |
| **perch** | leaderboards / standings (opt-in) | Phase 2 |
| **halo** | your profile — streak heatmap, stats, gleams, lume | MVP |
| **crest** | your public, verifiable, exportable card (the "skill passport" / resume) | Phase 3 |
| **field** | career & market — skill→job maps, company stacks, demand | Phase 3 |
| **ping** | broadcasts, reminders, newsletter — messages in | Phase 2 |
| **chart** | academic planning — subjects, timetable, PYQs, mock OA | Phase 3 |
